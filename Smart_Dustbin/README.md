# Smart Dustbin

This is a dustbin that automatically opens when a user approaches it. It also compresses garbage when the garbage is almost full to reduce number of times it is emptied.

## Parts
 - Ultrasonic sensor * 2
 - Stepper motor
 - l298 motor driver
 - NodeMCU esp8266
 - MG995 servo motor
 - Buck converter
 - Traffic leds module
 - Lead screw and travel screw
 - Dustbin
